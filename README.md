# homebrew-fonts

Homebrew tap for LXGW WenKai Nerd Font releases.

## Install

```bash
brew tap Yikai-Liao/fonts
brew install --cask font-lxgw-wenkai-nerd
brew install --cask font-lxgw-wenkai-mono-nerd
```

## Source

The casks in this tap download release assets from:

- https://github.com/Yikai-Liao/LxgwWenKaiNerdFont
